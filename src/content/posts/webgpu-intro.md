WebGPU 是下一代 Web 图形 API，旨在释放现代 GPU 的全部潜能。

## 与 WebGL 的区别

WebGL 基于 OpenGL ES，而 WebGPU 则是为了适应现代 GPU 架构（如 Vulkan, Metal, D3D12）而设计的。

- **更低的 CPU 开销**
- **更好的多线程支持**
- **计算着色器 (Compute Shaders)**

这是一个激动人心的时刻，Web 端的图形能力将迎来质的飞跃。
